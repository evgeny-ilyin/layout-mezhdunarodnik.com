Общественным транспортом

Пропускная система
  • Вход на территорию возможен только по пропускам (с паспортом). Заказать пропуск Вы можете, связавшись заранее с представителем Факультета по телефону +7 (499) 455-31-83.
  • Во время проведения общеакадемических Дней открытых дверей (одновременно во всех институтах Академии) проходит предварительная обязательная регистрация на странице мероприятия.
  • Во время проведения Дней открытых дверей Факультета международных отношений ИБДА РАНХиГС пропуск посетителей будет происходить по спискам, составленным на основе предварительной регистрации на мероприятие. Обязательно возьмите с собой паспорт!
